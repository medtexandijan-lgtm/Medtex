## Backend

Bu papkada Django backend joylashgan:

- `medical_crm/` - Django project konfiguratsiyasi
- `main/` - business logic, model, view va testlar
- `staticfiles/` - collectstatic natijalari
- `manage.py` - backend ichidan boshqarish uchun entrypoint

Root'dagi `manage.py` ham shu backend'ni ishga tushiradi.
